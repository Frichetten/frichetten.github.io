I Like To Work

Want to know what I've done? Well here you go! This is where I post about my side projects.

BURST Game Engine

This is a custom 2D game engine that I am developing in my free time. It is written in C/C++ and uses the SFML Graphical Libraries. I've really enjoyed this project because it lets me work with the low level components of the language, focussing on performance using threads and more.

easyPoll

easyPoll was a semester long project for IT326: Principles of Software Engineering. It was a massive endeavor, and took many long hours of work. The front end is Bootstrap, while the back end is Java Spring MVC. In addition there is a MySQL database.

RedBird Hacks Big Data Project

This was for the 2016 RedBird Hacks Hackathon at Illinois State University. The purpose of the hackathon was a full weekend of working on whatever you wanted to. I chose to use Python to perform an analysis on crash data. Working with more than 19 Million points of data, I learned a lot about the challenges of big data, and how to improve performance while writing Python. For a detailed look at my analyis please follow this link.

Image Difference Finder

This project is a personal favorite of mine. I got the idea while doing one of those "spot the difference" puzzles. I thought, why not develop a program that could find those differences between two panels? Thus, the Image-Difference-Finder was born. Simply insert an image, set the panel-divider, and it will hi-light the differences between the images. This is done primarily by reading pixel colors and checking for differences between a certain threshold.

Settlement Simulation Game

This project was my first attempt at building any sort of game. While I wouldn't recommend using Java as a game engine (due to performance issues), the Swing libraires were very helpful for drawing to the screen. In addition, I was really interested in understanding how to craft the game loop while maintaining optimum performance.
