How to Clone a Website

Something that consistently surprises me is when people remark on the fact that a phishing page looks "similar" or "identical" to the real page it is trying to emulate. A few weeks ago there were some minor rumblings about a phishing scheme that used a page that looked very similar tothe real Google login page (I have since tried to find those articles however the Google Docs worm fiasco has made searching for it dificult). It should not be revolutionary to think that a malicious actor would clone a webpage. In fact it's trivial to do so.
